# 🔌 PCB Design – Voltage Regulator

This project contains the complete PCB design of a **5V Voltage Regulator circuit**.
It includes schematic design, PCB layout, and Gerber files for fabrication.

---

## 🛠️ Tools Used

* KiCad (Schematic & PCB Design)

---

## ✨ Features

* Compact and efficient design
* Easy to manufacture
* Provides stable 5V output
* Suitable for embedded system applications

---

## 📁 Files Included

* Schematic Design
* PCB Layout
* Gerber Files

---

## 🚀 Applications

* Power supply circuits
* Microcontroller projects
* Electronics experiments
